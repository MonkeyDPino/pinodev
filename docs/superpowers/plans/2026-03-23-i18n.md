# i18n Language Switching Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add ES/EN language switching to the pinodev portfolio with localStorage persistence and a pill toggle in the header.

**Architecture:** Install react-i18next with the browser language detector plugin, initialize it in a single `src/i18n.ts` config file, store all translatable strings in flat JSON locale files, and add a `LanguageSwitcher` pill component to the header. Each component replaces hardcoded strings with `t('key')` calls; rich-text segments with inline markup use adjacent `t()` calls inside the original JSX wrappers.

**Tech Stack:** `i18next`, `react-i18next`, `i18next-browser-languagedetector`, TypeScript, SCSS

**Spec:** `docs/superpowers/specs/2026-03-23-i18n-design.md`

```ts
import i18n from "i18next";
import { initReactI18next } from "react-i18next";
import LanguageDetector from "i18next-browser-languagedetector";
import en from "./locales/en.json";
import es from "./locales/es.json";

i18n
  .use(LanguageDetector)
  .use(initReactI18next)
  .init({
    resources: {
      en: { translation: en },
      es: { translation: es },
    },
    fallbackLng: "en",
    detection: {
      order: ["localStorage", "navigator"],
      caches: ["localStorage"],
    },
    interpolation: {
      escapeValue: false,
    },
  });

export default i18n;
```

## Task 3: Create LanguageSwitcher component

**Files:**
- Create: `src/components/LanguageSwitcher/LanguageSwitcher.tsx`
- Create: `src/components/LanguageSwitcher/LanguageSwitcher.scss`

- [ ] **Step 1: Create `src/components/LanguageSwitcher/LanguageSwitcher.tsx`**

```tsx
import { useTranslation } from "react-i18next";
import "./LanguageSwitcher.scss";

export default function LanguageSwitcher() {
  const { i18n } = useTranslation();
  const currentLang = i18n.language?.startsWith("es") ? "es" : "en";

  return (
    <div className="language-switcher">
      <button
        className={`lang-btn${currentLang === "es" ? " active" : ""}`}
        onClick={() => i18n.changeLanguage("es")}
      >
        ES
      </button>
      <button
        className={`lang-btn${currentLang === "en" ? " active" : ""}`}
        onClick={() => i18n.changeLanguage("en")}
      >
        EN
      </button>
    </div>
  );
}
```

Note: `i18n.language` can be a full locale like `"es-CO"` from the browser — the `startsWith("es")` check handles this correctly.
